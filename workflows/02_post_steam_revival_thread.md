# Workflow 02: Post Steam Revival Thread

## Objective
Post a clear, well-written thread in the Knightfall Steam Community forums that tells new and returning players exactly when and where to show up to find a game. This is the permanent "home base" post that all other actions can link back to.

## Frequency
**One-time.** Post once, update only if the play schedule changes.

## Time Cost
**~15 minutes total** (copy template, fill in blanks, post)

## Required Inputs
- Steam account (logged in)
- Discord invite link from Rosefall or knightfallbr.com (get this in Workflow 05 first)
- Agreed play window time (set in Workflow 05 first)

## Do This After
Workflow 05 (Discord Setup) — you need the Discord link and play time before posting this.

## Steps

1. Go to the Steam Community Hub for Knightfall:
   `https://steamcommunity.com/app/1911390/discussions/`

2. Click **"Start a new discussion"**

3. Use the template below. Fill in the `[BRACKETS]`.

4. Post it. That's it.

---

## Post Template

**Title:**
```
[ACTIVE PLAYERS] When to play + Discord — Read this first
```

**Body:**
```
Hey everyone,

This game is still alive — it just needs everyone in the same place at the same time.

🕗 DAILY PLAY WINDOW
[TIME, e.g. 8:00 PM EST / 1:00 AM UTC]
Show up then and matches fill. Simple as that.

💬 DISCORD (where we coordinate)
[DISCORD INVITE LINK]
This is where sessions get announced, last-minute lobbies happen, and the community lives.

🆓 THE GAME IS FREE
If you haven't played in a while or have friends who haven't tried it — it costs nothing.
Share the store page: https://store.steampowered.com/app/1911390/

---

Why does this exist?
The game hit a matchmaking death spiral — players stop finding games, so they stop playing,
so there are fewer players. We're breaking that by concentrating everyone into one window.

See you on the road.
```

---

## After Posting
- Copy the URL of your post and save it in `.tmp/links.txt` — you'll reference it in other workflows
- Check back in 48 hours to reply to any comments (keep it brief)
- Do NOT create a second thread — one clear thread is better than several scattered ones

## Edge Cases
- If a thread already exists that's recent and well-written: don't create a competing one. Instead, comment on it with the Discord link and play time.
- If you can't post (Steam account too new): play the game for 5 minutes first to unlock community posting
