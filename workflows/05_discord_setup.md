# Workflow 05: Discord Setup & Play Schedule

## Objective
Join the existing Knightfall Discord communities, coordinate on a shared daily play window, and set up a recurring event so players always know when to show up. This is the foundation everything else depends on — do this first.

## Frequency
**One-time setup (~30 min).** After setup: check Discord for session announcements once per day (~1 min, optional).

## Time Cost
**~30 minutes** (one-time)

## Do This First
This workflow must be completed before Workflow 02 (Steam thread) and Workflow 04 (Creator Outreach), as both require the Discord invite link and play schedule.

## Required Inputs
- Discord account

## The Two Communities to Join

### 1. Rosefall Discord
- Description: Largest Knightfall community Discord
- How to find: Search "Rosefall Knightfall" in Discord's discovery, or find the invite in the Steam Community discussions for the game
- Why join: Biggest existing player pool — this is where the most organized sessions happen

### 2. knightfallbr.com Discord
- URL: `https://knightfallbr.com/discord`
- Description: Runs biweekly events with custom mods (solo mode, kill feed mod)
- Why join: More structured events, better for coordinating with admins

## Steps

### Part 1: Join and Orient (~10 min)
1. Join both Discords using the links above
2. Read the pinned messages and announcements in each
3. Note: what time do they currently schedule sessions? What channels are used?
4. Introduce yourself briefly in the general channel: "Hey, I'm trying to help grow the player count — happy to coordinate on play times if useful"

### Part 2: Coordinate a Daily Play Window (~15 min)
The goal is ONE shared daily window that both communities agree to, so players from both Discords show up at the same time.

1. In each Discord, post or DM the admins/moderators:
   ```
   Hey — I'm working on growing the daily player count. Would you be open to
   coordinating a shared daily play window with the other Knightfall Discord?
   Even just agreeing on a time (like 8pm EST) that we both announce would help
   matches fill more reliably.
   ```
2. Aim for a window between **7pm–10pm in the most active timezone** (check where most members are from — usually EST or EU)
3. Once a time is agreed: note it. This becomes the standard across all workflows.

**If communities won't coordinate:** Just pick the larger community's existing schedule and align with that. Don't force it.

### Part 3: Set Up a Recurring Event (~5 min)
If either Discord gives you moderator permissions or has a bot for events:
1. Create a recurring Discord event for the daily play window
2. Title: `Knightfall Daily Session — [TIME]`
3. Description: `Show up and matches fill. Free game on Steam: https://store.steampowered.com/app/1911390/`
4. Set to repeat daily

If you don't have permissions, ask an admin to set it up or do it manually each week.

## Outputs to Save
Once complete, save these in `.tmp/links.txt`:
```
Rosefall Discord invite: [LINK]
knightfallbr.com Discord invite: [LINK]
Agreed daily play time: [TIME + TIMEZONE]
```

These will be used in Workflow 02 (Steam post) and Workflow 04 (creator outreach).

## Edge Cases
- **Communities are unresponsive or inactive**: Use whichever one has active members and proceed independently with a play time
- **Communities conflict with each other**: Don't get involved in drama. Pick one and focus there.
- **Discord invite links expire**: knightfallbr.com/discord is a permanent redirect — use that one in public posts

## Notes
- You do NOT need to build a new Discord server. The communities already exist. Your job is to plug into them, not rebuild them.
- Being helpful in these communities (organizing sessions, welcoming new players) will naturally give you influence over time with minimal effort
