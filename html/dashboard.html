One-Time Setup Steps
  • 1
    Read the project goal
    Open README.md. Understand the mission, constraints, and phase overview before doing anything else.
    2 min
  • 2
    Set up the player count tracker
    Run python tools/track_player_count.py to confirm it works, then schedule it daily via Windows Task Scheduler. See workflows/01_track_player_count.md.
    5 min
  • 3
    Join the Discords — critical
    Follow workflows/05_discord_setup.md. Get invite links and agreed play times. Save them in .tmp/links.txt. Everything else depends on this step.
    30 min
  • 4
    Post the Steam revival thread
    Follow workflows/02_post_steam_revival_thread.md. Use the template. Fill in the Discord link and play time from Step 3.
    15 min
  • 5
    Run creator outreach
    Follow workflows/04_creator_outreach.md. Run the tool, pick 5–10 targets, send the templated message. Check for replies once a week after this.
    20 min
  • 6
    Post your first clip
    Follow workflows/03_tiktok_clip_pipeline.md. One clip to TikTok and/or YouTube Shorts. This is the highest-reach action available — start the habit now.
    20 min
